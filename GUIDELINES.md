# GIT REPO USAGE GUIDELINES

- General Guidelines for collaboration in this repository

1. Fork and maintain separate branch by each committer
2. Submit PRs for the main / release branch(s) (DO NOT push changes to `main` directly)
3. Keep private branch in sync on daily basis/ regularly
4. Follow README's in each subfolder for specific details in that sub-area
5. Maintain README's as appropriate. Provide enough details so that steps can be followed by other team members
6. Follow information about extending directory structure on [clusters/CLUSTER.md](clusters/CLUSTER.md)
